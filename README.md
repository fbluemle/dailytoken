## Getting Started

### Prerequisites

#### Environment

- A [React Native environment][4] with standard, up-to-date tools

### Installation

1. Install dependencies

```sh
yarn
```

2. For iOS, install pods

```sh
npx pod-install
```

3. Start the app

```sh
yarn android
# or
yarn ios
```

## Testing

Run the tests with:

```sh
yarn test
```
